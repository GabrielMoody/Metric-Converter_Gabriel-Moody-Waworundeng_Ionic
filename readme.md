# Metric Converter
### By: Gabriel Moody Wawowurundeng

Aplikasi berbasis mobile sederhana untuk melakukan konversi satuan. Satuan yang dapat dikonversi :

- Panjang (Meter)
- Massa (Gram)
- Waktu (Sekon)
- Suhu
